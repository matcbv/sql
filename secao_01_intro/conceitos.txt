O que é SQL?

A sigla SQL (Structured Query Lenguage) significa linguagem estruturada de consulta. É uma linguagem de banco de dados relacionais utilizada para criação de queries, responsáveis pela criação e manipulação de banco de dados.


SQL x MySQL

SQL é a linguagem e MysQL é o SGBD (Sistema Gerenciador de Bando de dados). Logo, MySQL é um software que gerencia nossos bancos de dados, enquanto SQL é a linguagem que vamos utilizar para manipular e criar os bancos.


O que é um bando de dados?

Bancos de dados é o local onde vamos armazenar os dados de nossa aplicação. Eles serão criados a partir da SQL, onde o MySQL nos auxiliará em seu gerenciamento. Um bando de dados possui entidades vitais paras eu funcionamento, como as tabelas, que são responsáveis por manter e organizar os dados.


Principais elementos de um bando de dados?

- Diagrama do Banco: Projeto do banco de dados, uma forma de documentação detalhando as partes fundamentais para seu funcionamento.
- Banco de dados: A entidade que guardará todos os elementos do banco.
- Tabelas: A categoria dos dados.
- Colunas: Tipos de informações que precisamos salvar.
- Dados: O dado final entregue pelo usuário.
