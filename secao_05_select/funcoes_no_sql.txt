Funções no SQL


MIN - Com a função MIN, obtemos o menor valor contido em uma coluna. Ex.:

    SELECT MIN(<column_name>) FROM <table_name>;


MAX - Com a função MAX, obtemos o maior valor contido em uma coluna. Ex.:

    SELECT MAX(<column_name>) FROM <table_name>;


Obs.: Utilizando MIN e MAX com valores não númericos, os organizará em ordem alfabética.


COUNT - Com a função COUNT, obtemos a contagem de valores contidos em determinada coluna. Ex.:

    SELECT COUNT(<column_name>) FROM <table_name>;


AVG - Com a função AVG, obtemos a média entre os valores contidos em determinada coluna. Ex.:

    SELECT AVG(<column_name>) FROM <table_name>;


SUM - Com a função SUM, obtemos a soma entre os valores contidos em determinada coluna. Ex.:

    SELECT SUM(<column_name>) FROM <table_name>;
